Applications


Tracking and monitoring

The Beacons are registered in the system and then attached to every asset and packages that needs to be tracked and monitored. Each beacon broadcasts a signal every few seconds indicating its presence along with the latest sensor reading in real time. The Gateways are deployed across the facility to cover maximum area. The signals transmitted from the beacons are received by the nearest Gateway. The Gateways maintain a constant link with the Nebulinx cloud services. Every new beacon signal received is passed securely to the Nebulinx Cloud Services. The Nebulinx Cloud Services are responsible for monitoring all the beacon activities, processing them and storing for later usage. We provide Nebulinx Visualization Tools for visualizing all the collected data in a variety of charts and diagrams for better understanding along with the analytical insights to help our customers take better strategical decisions and improve processes.



Location based services

The location aware Beacons are deployed at different locations in the facility. These beacons broadcast their location every few seconds. This creates a location aware ecosystem for our services for proximity marketing and indoor navigation. Nebulinx Cloud Services resolves the positioning requests by providing the precise indoor position and planning the most optimal indoor route. Nebulinx Visualization Tools can be used for visualizing all the positioning requests data for managing the facility in the most effective way and to get marketing insights. Event and location based triggers can be used to send notifications to the user and provide more location based information.
